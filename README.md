## Features

You can :

- log in
- sso
- log out
- sign up
- delete account
- add posts (You can add posts on the create page)
- delete posts
- like posts
- save posts
- Add a comment
- Delete a comment
- Product purchase(add to cart)
- View shopping cart
- Remove the product from the shopping cart
- View your profile information
- View about Instagram on the 'About Me' page
